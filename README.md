# Dự Án Du Lịch Việt Nam

Dự án này bao gồm một ứng dụng web và một ứng dụng di động để khám phá du lịch Việt Nam.

## Cấu Trúc Dự Án

- `web/`: Ứng dụng web được xây dựng bằng Next.js
- `app/`: Ứng dụng di động được xây dựng bằng Expo React Native

## Cách Chạy

### Web App
```bash
cd web
npm install
npm run dev
```
Mở http://localhost:3000

### Mobile App
```bash
cd app
npm install
npm run web  # Hoặc npm run android/ios
```

## Tính Năng
- Trang chủ với các điểm đến nổi bật
- Giao diện thân thiện cho web và mobile
